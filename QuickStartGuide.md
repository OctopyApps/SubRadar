# SubRadar — Быстрый старт

## Локальный режим

Скачайте приложение, откройте его и выберите **«Локально»**. Готово — данные хранятся прямо на вашем iPhone, интернет не нужен.

---

## Свой сервер (sync между устройствами)

### 1. Установите бэкенд на сервер

```bash
curl -fsSL https://raw.githubusercontent.com/OctopyApps/SubRadar-BackEnd/main/install.sh | sudo bash
```

Скрипт сам всё настроит и запустит. На вопрос про автозапуск ответьте `Y`.

### 2. Подключите iPhone

Откройте SubRadar → выберите **«Свой сервер»** → введите адрес:

```
http://ВАШ_IP:8080
```

Готово.
