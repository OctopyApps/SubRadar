# SubRadar — Карта экранов

```mermaid
flowchart TD
    A([Запуск приложения]) --> B[Онбординг\nВыбор режима хранения]

    B -->|Локально| C[Локальный режим]
    B -->|Общий сервер| D[Общий сервер]
    B -->|Свой сервер| E[Свой сервер]

    C --> MAIN
    D --> MAIN
    E --> MAIN

    subgraph MAIN [Основное приложение — Tab Bar]
        TAB1[Главная\nСписок подписок]
        TAB2[Аналитика\nРасходы, графики]
        TAB3[Настройки\nПрофиль, режим]
    end

    TAB1 --> F[Карточка подписки\nДетали]
    TAB1 --> G[Добавить\nНовая подписка]

    TAB2 --> H[Графики расходов\nПо месяцам, категориям]
    TAB3 --> I[Режим хранения\nСменить / настроить]
    TAB3 --> J[Уведомления]

    F --> K[Редактировать\nИзменить данные]
    G --> L[Поиск сервиса\nNetflix, Spotify…]
    H --> M[Прогноз расходов\nНа год вперёд]
```

## Легенда

| Цвет | Группа экранов |
|---|---|
| Фиолетовый | Онбординг |
| Зелёный | Режимы хранения |
| Синий | Tab bar (основные вкладки) |
| Коралловый | Экраны подписок |
| Янтарный | Аналитика |
| Серый | Настройки |
