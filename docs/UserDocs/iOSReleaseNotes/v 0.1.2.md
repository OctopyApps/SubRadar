# Backend v0.1.2

## Что нового

### Многопользовательская поддержка
Все данные теперь привязаны к конкретному пользователю через JWT. Один бэкенд может обслуживать нескольких пользователей независимо — подписки, категории, валюты и теги каждого пользователя изолированы друг от друга.

### Новые эндпоинты: категории и валюты
Пользователи могут создавать собственные категории и валюты — они синхронизируются через сервер.

```
GET    /categories
POST   /categories
DELETE /categories/{id}

GET    /currencies
POST   /currencies
DELETE /currencies/{id}
```

### Миграция базы данных
Добавлены две новые таблицы: `categories` и `currencies`. При обновлении с v0.1.1 миграция применяется автоматически при старте сервера — никаких ручных действий не требуется.

## Обновление

```bash
curl -fsSL https://raw.githubusercontent.com/OctopyApps/SubRadar/main/install.sh | sudo bash
```

Или скачай бинарник вручную со [страницы релизов](https://github.com/OctopyApps/SubRadar/releases/tag/backend/v0.1.2).

# iOS v0.1.2

## Что нового

### Подключение к бэкенду
Приложение теперь полноценно работает в режимах **Общий сервер** и **Свой сервер** — авторизация, синхронизация подписок, тегов, категорий и валют через API.

### Авторизация
- Вход по email и паролю
- Регистрация нового аккаунта — после успешной регистрации сразу переходишь в приложение без повторного входа
- Self-hosted вход по секретному ключу

### Синхронизация пользовательских категорий и валют
Категории и валюты которые пользователь создаёт в настройках теперь сохраняются на сервере и доступны на всех устройствах. Встроенные категории (Развлечения, Работа, Здоровье, Обучение, Другое) и валюты (RUB, USD, EUR и др.) по-прежнему захардкожены на клиенте и не требуют запроса к серверу.

## Технические изменения

- `Subscription`, `Tag`, `AppCategory`, `AppCurrency` — добавлен `Codable` для работы с API
- `RemoteStorageService` — полная реализация всех методов протокола `StorageService`
- `APIClient` — переиспользуемый HTTP-клиент с поддержкой JWT, ISO8601 дат и парсингом ошибок сервера
- `AuthService` — единый сервис для всех auth-запросов (`/auth/login`, `/auth/register`, `/auth/self-hosted`)

## Требования

- iOS 17+
- Для режима синхронизации: SubRadar Backend v0.1.2+
